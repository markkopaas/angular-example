Simplest possible example of angularjs

Prebuilt version:

	(https://markkopaas.github.io/angular-example/build)

Install and build:

	npm i

Use any http server, for example this tiny little gem:

	npm i http-server -g
	http-server

Now you can browse your app:

	(http://localhost:8080)
	
or
	
	(http://localhost:8080/build)
